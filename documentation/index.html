AngelOn API

AngelOn API is the backend implementation for the AngelOn project.

REST services provided using Nest.js framework.

Deployment

Click Me

AngelOn currently uses MongoDB database through @nestjs/mongoose. .

We recommend you to use MongoDB Atlas. Their free tier service will allow you to deploy MongoDB using the provider and region of your choice between Amazon AWS, Google Cloud and Microsoft Azure.

If you are going to deploy MongoDB manually, be sure to a replica set.

Installing

$ npm install

Running

# development
$ npm run start

# production mode
$ npm run start:prod

Test

$ npm run test
