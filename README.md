## Description

AngelOn API is the backend implementation for the AngelOn application.

We are using Nest.js framework providing us Node.js, TypeScript, Fastify, Prisma.

## Installation

```bash
$ npm install
```

## Deployment

    [   ] Docker instructions

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
